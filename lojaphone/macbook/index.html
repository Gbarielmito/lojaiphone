Design

Uma composição radicalmente original.

O fone de ouvido foi completamente reimaginado. De almofada para dossel, os AirPods Max são projetados para um ajuste que cria a vedação acústica ideal para muitas cabeças diferentes formas - mergulhando você totalmente em cada som.

  • O dossel abrangendo a faixa de cabeça é feita de uma malha de malha respirável, distribuindo o peso para reduzir pressão na cabeça.

  • O estrutura de aço inoxidável is wrapped with a soft-to-the-touch material for a remarkable combination of strength, flexibility, and comfort.

  • Braços telescópicos Suavemente estender e permanecer onde você os coloca, para um ajuste e vedação consistentes.

  • O copos de alumínio lindamente anodizados Característica A mecanismo revolucionário que permite que cada copo gire independentemente e equilibrar a pressão.
